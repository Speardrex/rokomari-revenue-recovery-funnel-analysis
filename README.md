# 📊 Rokomari E-Commerce Growth & Funnel Audit

## 🚀 Executive Summary
This project is a proactive, data-driven Growth Audit of Rokomari, Bangladesh's leading e-commerce platform for books and electronics. The goal of this analysis was to identify UI friction points, model the resulting Revenue Opportunity Gap, and design an ICE-prioritized 60-day roadmap for revenue recovery.

**Key Outcome:** Identified that 82% of site-wide revenue leakage is concentrated in just 20% of product categories, projecting a 10% pure-margin revenue recovery by targeting specific UX bottlenecks.

## 🛠️ Tools & Methodology
* **Analytical Approach:** Pareto Analysis (80/20 Rule), Funnel Drop-off Modeling, ICE Scoring Framework.
* **Tech Stack:** Excel / SQL / Python / Presentation Design
* **Key Metrics Tracked:** Average Order Value (AOV), Exit Rate, Dwell Time, Rage Clicks, Loss Score.

## 📉 Key Business Insights
1. **The "Vital Few":** High-AOV categories (Technical Books & Electronics) suffer from a high Loss Score due to UI friction, despite high user intent.
2. **Technical Friction:** Identified **961 Rage Clicks** in the checkout flow for Books, indicating a severe drop-off point.
3. **Trust Gap:** A **47.5% Exit Rate** on high-ticket items pointed to cognitive overload and a lack of social proof.

## 🗺️ 60-Day Strategic Roadmap
*Developed a phased intervention plan prioritized by Impact, Confidence, and Ease (ICE).*

| Phase | Strategy | Target Metric | ICE Score |
| :--- | :--- | :--- | :--- |
| **Days 1-20** | Frictionless Checkout (Books) | Reduce Rage Clicks | 9.3 |
| **Days 1-20** | Authority Copywriting Anchor | Reduce Exit Rate | 8.8 |
| **Days 21-45** | Cognitive Load Reduction (UI) | Improve Dwell Time | 8.0 |
